# 音声ファイル配置場所

このフォルダに以下の音声ファイルを配置してください：

## 必要なファイル

- `jump.mp3` - ジャンプ音
- `coin.mp3` - コイン取得音
- `goal.mp3` - ゴール到達音（クリア時）
- `gameover.mp3` - ゲームオーバー音
- `bgm.mp3` - ゲーム中のBGM（ループ再生）

## 推奨形式
- MP3 または OGG形式
- ファイルサイズ：各100KB以下推奨

## フリー素材サイト
- [効果音ラボ](https://soundeffect-lab.info/)
- [DOVA-SYNDROME](https://dova-s.jp/)
